# 🤖 FAQ Chatbot using Streamlit & Python

An interactive, smart **FAQ Chatbot** web application built using Python and the **Streamlit** framework. This bot is designed to understand user queries about specific domains (like college, product, or services) and provide instant, accurate answers from a predefined FAQ knowledge base.

---

## 🚀 Features

* **Instant Responses:** Provides real-time answers to frequently asked questions.
* **Smart Matching:** Uses text matching / NLP (Natural Language Processing) to understand user intent even if the wording is slightly different.
* **Chat Interface:** Clean, modern, and interactive chat UI utilizing Streamlit's native chat elements (`st.chat_message`).
* **Fallback Mechanism:** Politely informs the user if the question is outside the knowledge base.

---

## 🛠️ Tech Stack

* **Frontend: Streamlit
* **Language:** Python
